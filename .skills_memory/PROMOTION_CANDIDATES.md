# Promotion Candidates

Records use the following required fields:
- Candidate ID
- Record ID
- Target Skill
- Proposed Change Summary
- Target Files
- Expected Improvement
- Risk Type
- Severity
- Occurrence Count
- Transferability Score
- Impact Score
- Testability Score
- User Correction Strength
- Safety Risk
- Attribution Confidence
- Promotion Score
- Promotion Decision
- Reason
- Eligible Target
- Created At
- Status
- Evaluation Plan
- Rollback Plan
